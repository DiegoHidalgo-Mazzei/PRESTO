Política de Cookies

Las Cookies son ficheros con una pequeña cantidad de datos que se utilizan comúnmente como identificadores únicos anónimos. Estos se envían a vuestro navegador desde los sitios web que visitáis y se almacenan a la memoria interna de vuestro dispositivo.

Este servicio no utiliza estas "cookies" explícitamente. Aun así, la aplicación puede utilizar código de terceros y bibliotecas que utilizan "galletas" para recopilar información y mejorar sus servicios. Podéis aceptar o rechazar estas galletas y saber cuándo se envían las galletas al dispositivo. Si decidís rechazar nuestras galletas, es posible que no podáis utilizar algunas partes de este servicio.

La App es nativa, NO GENERA NI ALMACENA NINGÚN TIPO DE COOKIES.
